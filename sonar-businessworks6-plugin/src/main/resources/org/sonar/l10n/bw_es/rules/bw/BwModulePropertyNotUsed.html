Los atributos de los recursos compartidos y propieades de las actividades deberían ser configurados utilizando Application, Module y Process Properties. Cuando dichos recursos o actividades se eliminan, muchas veces se quedan las variables, lo que dificulta el mantenimiento de los proyectos.

Por defecto, System Module Properties BW.*, están excluidas, así como cualquier que cumpla el patrón.
